# \<kemet-drawer>

## Installation
```bash
npm i @kemet/kemet-scroll-link
```

## Usage
```js
import '@kemet/kemet-scroll-link/kemet-scroll-link.js';;
```

```html
<kemet-scroll-link>
  <button>Scroll to Top</button>
</kemet-scroll-link>
```

[Click here](http://kemet.online/scrolllink) for more docs.
